Desenvolvido com o micro-framework FLASK no Python

O Flask é um micro-framework que permite criar sites, aplicativos web e APIs de forma rápida e eficiente, com a possibilidade de adicionar funcionalidades gradualmente, como suporte a banco de dados, autenticação, templates entre outras coisas mais.

Empresas como Zillow, Patreon, Reddit, Trivago, Pinterest e várias outras usam Flask em seus sites. Além disso, diversas empresas de ponta usam Flask em sua pilha, como Samsung, Netflix, Uber, Airbnb e outras.

Neste curso, os alunos do Curso Técnico em Ciência de Dados da E.E. Dr Ângelo Mendes de Almeida aprenderam a criar este site, suas múltiplas páginas, a integração com um banco de dados e formulários de cadastro, login e contato usando o Flask em seu desenvolvimento e o consumo de APIs em Python no back-end.

Todo o projeto foi desenvolvido em sala de aula com a participação dos alunos.
